# 上海大观园数字孪生导览系统

> 🏯 基于微信小程序的《红楼梦》主题园林数字导览  
> 🎓 交互设计留学作品集项目

---

## 项目简介

让每一位游客都能「走进」红楼梦——通过数字孪生技术，将上海大观园的物理空间与《红楼梦》的文学故事深度关联，解决"有景无故事"的游览痛点。

### 核心功能

| 功能 | 说明 | 状态 |
|------|------|:--:|
| 🗺️ 2D园区导览地图 | 基于手绘地图数字化，支持缩放/平移/GPS定位 | ✅ |
| 📍 GPS实时定位 | 自动检测用户位置，近场推送景点故事 | ✅ |
| 📖 景点故事卡片 | 每个景点关联红楼梦人物与情节，3层展开 | ✅ |
| 🎬 87版视频片段 | 在对应景点播放经典剧情（30-90秒） | ✅ |
| 🎙️ 语音讲解 | AI配音，每条≤2分钟 | ✅ |
| 🗺️ 3条主题路线 | 宝黛爱情线/十二金钗探秘/元妃省亲盛典 | ✅ |
| 📱 AR实景识别 | 扫描建筑自动识别并叠加信息标签 | ✅ |
| 🏆 打卡成就系统 | 游览/路线/收藏达成条件解锁成就徽章 | ✅ |
| ❤️ 收藏与分享 | 收藏景点、生成游览卡片分享 | ✅ |
| 🔗 人物关系图谱 | 可交互的红楼梦人物关系图 | P1 |

## 快速开始

### 环境要求

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 微信小程序 AppID（测试号或正式号）
- Node.js（可选，用于资源处理脚本）

### 运行步骤

1. 下载微信开发者工具
2. 打开工具 → 导入项目 → 选择 `daguanyuan-miniapp/` 目录
3. 填入 AppID（可使用测试号）
4. 点击编译预览

### 配置 CDN

视频和音频资源需要托管在 CDN 上：

1. 将视频/音频文件上传至腾讯云COS
2. 更新 `data/spots.json` 中的 `video` 和 `audio` 字段为CDN外链
3. 在微信小程序后台配置 `uploadFile` 和 `downloadFile` 域名白名单
